一.创建本地仓库

创建一个本地仓库(文件夹),将该文件夹初始化为git仓库，即在该文件夹下点击鼠标右键(git bash here),然后使用 git init 命令，这样，初始化仓库就建好了

新克隆下来的代码默认master和origin/master是关联的，也就是他们的代码保持同步
但是origin/dev分支在本地没有任何的关联，所以我们无法在那里开发
